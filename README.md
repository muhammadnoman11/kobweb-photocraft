# 🎨 PhotoCraft

**PhotoCraft** is a modern, web-based photo editor built with **Kotlin, Compose for Web, Kobweb, and Fabric.js**, offering professional-grade image editing directly in your browser.

---

## 🚀 Features

### 🖼️ Image Editing

- Drag & drop image upload
- Crop and resize images
- Real-time preview

### 🎛️ Filters & Adjustments

- Built-in filter presets (Vivid, Noir, Sepia, Vintage, etc.)
- Advanced adjustments:
    - Brightness
    - Contrast
    - Saturation
    - Hue
    - Blur
    - Shadows & highlights
    - Temperature & tint
    - Vignette & noise

### 🔤 Text Editor

- Add custom text to the canvas
- Multiple fonts (Google Fonts integrated)
- Full control over:
    - Font size
    - Weight & style
    - Alignment
    - Stroke & shadow
    - Letter spacing

### 🧩 Stickers & Shapes

- Emoji stickers 🎉🔥❤️
- Decorative symbols
- Arrows and shapes
- Fully draggable, scalable, and rotatable

### 📐 Canvas & Layers

- Layer management system
- Lock / hide layers
- Undo & redo support *(partially implemented)*
- Object selection tracking

### 🖌️ Drawing Engine

- Powered by Fabric.js for advanced canvas manipulation

## ⚙️ Setup Kobweb CLI (Required)

Before running the project, install the Kobweb CLI.

### 1. Download Kobweb CLI

Download the latest version from:  
https://github.com/varabyte/kobweb-cli/releases

### 2. Extract the archive

After downloading, extract the ZIP file. Inside, you will find a `bin` folder.

### 3. Add Kobweb to Environment Variables

#### Windows:

1. Copy the path of the `bin` folder
2. Open **Environment Variables**
3. Edit the **Path** variable
4. Click **New** and paste the path
5. Click **OK**

### 4. Verify installation

```bash
kobweb --version
```

## ▶️ Run PhotoCraft

### In your terminal:

### 1. Navigate to the site directory

```bash
cd site
```

### 2. Run the project

```bash
kobweb run
```

### 3. Open in your browser

```
http://localhost:8080
```
